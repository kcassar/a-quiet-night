FIRST CPAP SETUP CHECKLIST
==========================

Educational only — please follow your clinic's instructions for your specific device.

Before you start
[ ] Wash hands and a clean surface to assemble on
[ ] Read the manufacturer's quick-start guide
[ ] Check that the SD card is in the machine

Mask
[ ] Try the mask on while standing, then sitting on the bed
[ ] Adjust headgear evenly; finger-snug, not tight
[ ] With the machine on, gently lift the cushion off your face and let it
    re-seat naturally
[ ] Test in your usual sleeping position

Machine
[ ] Confirm the pressure setting matches what your clinic prescribed
[ ] Enable ramp if pressure feels high at first
[ ] Set humidifier to a comfortable starting point
[ ] If using heated tubing, set it warm enough to avoid hose condensation

Routine
[ ] Have water nearby for the humidifier (distilled if possible)
[ ] Charge any battery accessories
[ ] Plan a daily wipe and weekly wash for the mask
[ ] Keep the prescription/letter near the machine for travel

If anything doesn't feel right
[ ] Make a note (date, what happened, when)
[ ] Bring the notes to your next clinic appointment

This checklist is informational only and does not replace clinical advice.
